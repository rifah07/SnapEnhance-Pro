About the Project

SnapEnhance Pro is an advanced image processing platform that allows users to apply various AI-powered effects to their photos. The system features:

  • 17 different image effects
  • User authentication system
  • Image processing history
  • Responsive design for all devices
  • Secure file handling

The project was developed as part of our academic coursework at Jagannath University, combining our knowledge of cloud computing, computer vision, web development, and system design.
